- 고차 함수는 함수를 값처럼 다루는 함수를 뜻 함 - 일반적으로 다음 두 가지 중 하나를 만족하면 고차 함수라 함 - 함수를 인자로 받는 함수(메서드) - 함수를 반환하는 함수 (메서드) #### 함수를 인자로 받는 경우 ```java // 함수(람다)를 매개변수로 받음 static void calculate(MyFunction function) { // ... } ``` #### 함수를 반환하는 경우 ```java static MyFunction getOperation(String operator) { // ... return (a, b) -> a + b; } ``` - 즉, 매개변수나 반환 값에 함수(또는 람다)를 활용하는 함수가 고차 함수 - 자바에서 람다(익명 함수)는 함수형 인터페이스를 통해서만 전달할 수 있음 - 자바에서 함수를 주고받는다는 것은 `함수형 인터페이스를 구현한 어떤 객체(람다든 익명 클래스든)를 주고받는 것`과 동의어임 (함수형 인터페이스는 인터페이스이므로 익명 클래스, 람다 둘 다 대입할 수 있음, 하지만 실질적으로 함수형 인터페이스에는 람다를 주로 사용)
